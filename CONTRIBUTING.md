# Contributing

Thanks for contributing to `wisp-core`.

## Before You Start

- Open an issue (or comment on an existing issue) for non-trivial changes.
- Keep pull requests focused and small where possible.
- Add or update tests when behavior changes.

## Development

- Build: `swift build`
- Run tests: `swift test`

## Commit Sign-off (Required)

By contributing, you certify that you have the right to submit the work under
the project license and agree that your contribution is licensed under the
repository's `LICENSE` file.

Add this trailer to every commit:

`Signed-off-by: Your Name <you@example.com>`

You can do this automatically with:

`git commit -s`

## Pull Request Checklist

- Code builds locally.
- Tests pass locally.
- Documentation is updated if behavior or interfaces changed.
- Commits are signed off (`-s`).
