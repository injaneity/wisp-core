# Code of Conduct

## Our Standards

This project expects all participants to:

- Be respectful and constructive.
- Assume good intent and give actionable feedback.
- Avoid harassment, discrimination, and personal attacks.

## Scope

This Code of Conduct applies in project spaces, including issues, pull
requests, discussions, and other community channels tied to this repository.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, issues, and
other contributions that violate this Code of Conduct.

## Reporting

Report unacceptable behavior privately to the maintainers via security contact
or repository owner contact information.
